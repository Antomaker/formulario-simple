# Formulario Simple – Spring Boot + JS  
Demo didáctica de 2 h: formulario que envía y lista mensajes sin recargar la página.

## Arranque rápido
```bash
./mvnw spring-boot:run
```
Abre → http://localhost:8080

## Endpoints
- `GET  /api/mensajes`      – lista mensajes  
- `POST /api/mensajes`      – crea mensaje  
- `GET  /api/mensajes/code` – devuelve el código fuente del controller

## Ejercicio
Añade campo **edad** (≥ 1) back + front y recarga.

## Stack
Java 17 | Spring Boot 3 (web) | Bootstrap 5 | JS vanilla | Map en memoria
